🚀 快速开始（5分钟）
================================================================================

第1步：启动 Ollama
   $ ollama serve

第2步：拉取模型（另一个终端）
   $ ollama pull llama2

第3步：配置 .env 文件
   LLM_PROVIDER=ollama
   OLLAMA_MODEL=llama2
   DEFAULT_MODEL_NAME=llama2

第4步：验证配置
   $ python test_system_upgrades.py

完成！现在可以使用本地 Ollama 模型了。

================================================================================
💻 使用示例
================================================================================

1️⃣ 获取实时行情
   from realtime_data_fetcher import realtime_fetcher
   quote = realtime_fetcher.get_realtime_quote('000858')
   print(f"当前价格: {quote['price']} 元")

2️⃣ 获取当日K线
   today = realtime_fetcher.get_today_data('000858')
   print(f"收盘价: {today['close'].iloc[0]}")

3️⃣ 验证数据新鲜度
   fresh = realtime_fetcher.verify_data_freshness('000858')
   print(f"数据状态: {fresh['status']}")

4️⃣ AI分析会自动使用配置的 LLM
   from news_flow_agents import NewsFlowAgents
   agents = NewsFlowAgents()
   result = agents.sector_impact_agent(...)

================================================================================
🔄 切换 LLM 提供商
================================================================================

修改 .env 文件中的 LLM_PROVIDER：

✅ Ollama（本地，免费，离线使用）
   LLM_PROVIDER=ollama
   OLLAMA_MODEL=llama2

✅ DeepSeek（云端，便宜）
   LLM_PROVIDER=deepseek
   DEEPSEEK_API_KEY=sk_xxx

✅ OpenAI（云端，较贵）
   LLM_PROVIDER=openai
   OPENAI_API_KEY=sk_xxx

✅ Qwen（云端，便宜）
   LLM_PROVIDER=qwen
   QWEN_API_KEY=xxx

✅ 自定义 API
   LLM_PROVIDER=custom
   CUSTOM_BASE_URL=https://your-api.com/v1
